Efficient Door

Are you tired of vanilla doors because you can only open one door at a time? Now with this script, you can open two connecting doors at the same time. By default interacting with the door only can interact with one door at the same time. Now with this script, you can open two connecting doors at the same time. Efficient Door is a simple script that makes interacting with doors more efficient. This script can use in all vanilla doors. Sneaking can disable the connected door feature.
